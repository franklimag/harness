# 学习计划

## 总体目标

系统学习主流 Agent Harness 框架的设计思想、核心实现和最佳实践。

## 学习路线

### 阶段一：调研与概览

- [ ] 收集热门 Agent Harness 项目
- [ ] 了解各项目的定位和核心功能
- [ ] 对比分析各项目的优劣势

### 阶段二：深入学习

- [ ] 选择 2-3 个重点项目进行深入研究
- [ ] 阅读源码，理解核心架构
- [ ] 尝试动手实践，运行示例

### 阶段三：实践总结

- [ ] 基于学习成果进行实践项目
- [ ] 输出学习总结和对比文档
- [ ] 探索自定义 Agent Harness 的可能性
