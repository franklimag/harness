# Agent Harness 学习资源库

收集热门的 Agent Harness 相关的仓库信息，并记录学习计划和学习笔记。

## 项目结构

```
.
├── README.md                # 项目说明
├── repos/                   # 热门 Agent Harness 仓库信息
│   └── README.md            # 仓库汇总列表
├── plans/                   # 学习计划
│   └── README.md            # 学习计划总览
└── notes/                   # 学习笔记
    └── README.md            # 笔记索引
```

## 目录说明

### repos/ — 仓库信息

收集和整理热门 Agent Harness 相关项目的信息，包括：
- 项目名称、链接、Star 数
- 核心功能描述
- 技术栈
- 适用场景

### plans/ — 学习计划

制定和跟踪学习计划：
- 学习目标
- 时间安排
- 里程碑

### notes/ — 学习笔记

记录学习过程中的笔记：
- 源码阅读笔记
- 架构分析
- 实践心得
- 对比总结

## 什么是 Agent Harness？

Agent Harness 是用于构建、测试、评估和运行 AI Agent 的框架和工具集。它们通常提供：
- Agent 执行环境（沙箱）
- 工具调用接口
- 评估基准（Benchmark）
- 编排和调度能力
- 可观测性和调试支持

## 如何贡献

1. 在 `repos/` 中添加新发现的仓库信息
2. 在 `plans/` 中更新学习计划
3. 在 `notes/` 中添加学习笔记
