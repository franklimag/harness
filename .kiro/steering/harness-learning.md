# Agent Harness 学习工程 Steering

## 项目目标

本仓库用于收集热门 Agent Harness 相关仓库的信息，并记录学习计划和学习笔记。
**不 clone 目标仓库的源码**，只记录信息和笔记。

## 工作约定

### 目录结构
- `repos/` — 各仓库的信息卡片（每个项目一个 md 文件）
- `docs/` — 参考资料、对比分析
- `plans/` — 学习计划
- `notes/` — 学习笔记

### 项目收录规则
- **项目由维护者直接指定**，不自行搜索添加
- 每个项目基于 `repos/_template.md` 模板创建信息卡片
- 信息通过搜索公开资料获取，不 clone 仓库

### 笔记规范
- 每个主题一个独立 Markdown 文件
- 包含：概念说明、架构分析、学习记录
- 使用表格追踪学习进度
- 文件命名：`YYYY-MM-DD-主题.md`

### 更新流程
- 新增项目时基于模板创建 `repos/<project>.md`
- 学习新知识时在 `notes/` 下添加笔记
- 新的参考资料添加到 `docs/references.md`
- 对比发现更新到 `docs/comparison.md`

## 协助要求

1. 收录新项目时，搜索公开信息填充模板
2. 保持中文撰写
3. 不自行添加项目，由用户指定
4. 专注于 Agent Harness 相关内容（不含通用 AI 框架）
